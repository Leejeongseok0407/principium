git 명령어 저장 git : 기초명령어 출력 add : 스테이지에 올림 pull : 원격 저장소에 올라간 파일을 불러오기(당겨오기) push : 원격 저장소에 커밋 업로드 commit : 세이브, 스테이지 영역 파일들을 가지고 커밋(세이브) 만들수 있음 clone : 원격 저장소 저장
